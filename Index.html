Projet de site Web simple

Visiter le site de la faculté

Ce site contient la liste des modules que j'ai étudiés dans mon université au formation liscence et un formule d'inscription auxquelles vous pouvez vous inscrire, ainsi que l'administrateur auquel il peut voir les informations des nouveaux étudiants qui se sont inscrits. Ce simple site web est construit avec HTML, CSS, JS, PHP, BOOTSTRAP et MYSQL, vous pouvez trouver dans le code de ce site beaucoup de choses comme: comment créer une barre de navigation ou un jumbotron, un pied de page (Footer) ou comment créer un formulaire et le manipuler avec JS comme vous le souhaitez et sauvegardé les données que vous obtenez du formulaire dans la base de données, et vous pouvez également entrer dans la session de l'administrateur, la base de données de ce site contient deux tables, la première est destinée aux nouveaux étudiants qui se sont inscrits dans Formulaire d'inscription et le deuxième table pour l'administrateur, et tu vas trouver beaucoup plus autre chose, c'est un projet simple avec lequel vous pouvez vous entraîner pour améliorer vos compétences en développement web.
